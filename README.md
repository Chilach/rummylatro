#Rummy-Latro

##Descripción 
- Es un proyecto inspirado en Balatro hecho para una experiencia multijugador con las reglas de el Rummy.
- La principal intención de este proyecto es la construcción de un portfolio para la obtención de un trabajo.
- Escrito en C/C++ y de código abierto para que cualquier persona pueda gozar de un Rummy-Latro como lo hago yo.

##Comentarios personales
- Este es el inicio de un proyecto personal, hay muchas cosas que pueden salir mal, sin embargo las que más importan son las que tenemos en frente de nuestros ojos y dedicamos con amor al mundo desde la contemplación y gracias de la existencia.
- La única parte del proyecto escrito en mi lengua materna es este README.md, lo demás (incluso el código) está escrito en inglés para la comodidad de la comunidad.
